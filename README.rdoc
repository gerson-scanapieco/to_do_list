== To-Do List

Aplicativo para gerenciamento de Listas de Tarefas. Os usuários podem criar Listas de Tarefas
públicas (visíveis para todos os usuários) ou privadas (visível apenas para quem a criou), e
também adicionar Listas de Tarefas aos Favoritos.

* Ruby 2.2.1

* PostgreSQL 9.3.6

* Twitter Bootstrap 3
